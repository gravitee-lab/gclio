# `gclio` The Gravitee Dev Team CLI


* install :
```bash
# --- #
npm i
npm run build
sudo npm i -g ./
gclio --version
gclio --help

```

* uninstall :

```bash
# --- #
sudo npm uninstall -g ./
sudo npm unlink
gclio --version

```




## Build from source

* git clone this repository, and run :

```bash
# --- #
npm i
npm run build
```
